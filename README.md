Scaffold structures - predefined or custom

### Install
```
npm install takeoff-cli -g
```

### Usage
```
  Usage
    $ takeoff <station>

  If no station is given a interactive cli prompt will ask you which station you want.

  Examples
    $ takeoff
    - or
    $ takeoff node-module
```
